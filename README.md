### ðŸš€ Deployment Guide (GitHub Method - Recommended)

GitHub se connect karne par aapki website automatic update hogi aur koi error nahi aayega.

#### Step 1: Code Download Karein
1. Upar **"Export"** button dabayein > **"Download Zip"**.
2. Zip file ko apne computer/phone mein **Extract (Unzip)** karein.

#### Step 2: GitHub Par Dalein
1. [GitHub.com](https://github.com) par login karein.
2. **(+)** Icon dabayein > **New Repository**.
3. Repository ka naam dalein (e.g., `lords-tailor`) aur **Create Repository** karein.
4. Agle page par **"uploading an existing file"** link par click karein.
5. Apne computer se saari files wahan **Drag & Drop** karein.
6. Niche **"Commit changes"** button dabayein.

#### Step 3: Netlify Se Connect Karein
1. [Netlify.com](https://app.netlify.com) par login karein.
2. **"Add new site"** > **"Import from existing project"**.
3. **GitHub** select karein.
4. Apni `lords-tailor` repository select karein.
5. **Deploy Site** par click karein.
